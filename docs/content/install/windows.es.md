---
title: "Windows"
---

Tu puedes instalar este programa descargando el archivo [release][2] named
`Simple-DHCP-Server-Windows-X.Y.Z.zip`. Descomprimiendo el archivo zip,
encontraras un archivo ejecutable `.exe` dentro.

You might need to unblock the file:

![Properties -> Unblock -> OK](/img/windows-unblock.png)

[2]: https://github.com/niccokunzmann/simple_dhcp_server/releases
